# Expense Tracker (MERN)

> Full stack expense tracker

## Front-End - React hooks using context api
## Back-End - Node.js, Express.js & MongoDB

### Author : Hammad Ali

#### What you need to run this code
1. Node (13.12.0)
2. NPM (6.14.4) or Yarn (1.22.4)
3. MongoDB (4.2.0)


####  How to run this code
 npm install
 cd client npm install
 cd ..
 
 # Run front and backend
 npm run dev
 
 # Backend only
 npm run server
 
 # Frontend only
 npm run client
 
 # Build client
 cd client
 npm run build
 
 # Prod
 npm start
```
